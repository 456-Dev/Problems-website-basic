## Animations

### 1. Fade In
- Used on page load
- 0.3s duration
- All major sections

### 2. Slide Up
- Used on hero title
- 0.3s duration
- Slides up 10px

### 3. Pulse (Subscribe Button)
- Continuous animation
- Fades in/out
- Draws attention

### 4. Blink (Loading Text)
- Old-school terminal effect
- 1s interval
- Text appears/disappears

### 5. Spin (Loading Box)
- Inner box rotates
- 1s duration
- Continuous
